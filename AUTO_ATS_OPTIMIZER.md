## 🚀 How to Use Auto ATS Optimizer

### Step-by-Step Process

1. **Navigate to Job Posting**
   - Visit any job board or company career page
   - Find a job posting you're interested in
   - Keep the job page open in your current tab

2. **Open Auto ATS Optimizer**
   - Click the Job Copilot extension icon
   - Select "Auto ATS Optimizer" from the menu
   - The tool automatically detects if you're on a job page

3. **Extract Job Information**
   - Click "Extract Job Description from Current Page"
   - Watch as the system analyzes and extracts job details
   - Review the detected job title, company, and description

4. **Upload Your Resume**
   - Click "Browse Files" to upload your current resume
   - Supported formats: PDF, DOCX, TXT (up to 5MB)
   - System validates and processes your resume content

5. **Optimize for ATS**
   - Select your preferred ATS aggressiveness level
   - Click "Optimize Resume for ATS"
   - Review the detailed improvements made

6. **Download Optimized Resume**
   - Click "Download Optimized Resume"
   - Save the ATS-optimized version
   - Use for your job application

## 🆘 Troubleshooting

### Common Issues

**Extraction Failures:**
- Ensure you're on a valid job posting page
- Try refreshing the page and retry extraction
- Use manual entry mode as fallback
- Check if the site uses dynamic loading

**Low Optimization Scores:**
- Provide more detailed job descriptions
- Include specific skills and requirements
- Consider higher aggressiveness level
- Review original resume completeness

**File Upload Problems:**
- Verify file format is supported
- Check file size is under 5MB
- Ensure file isn't corrupted
- Try different browser if issues persist
