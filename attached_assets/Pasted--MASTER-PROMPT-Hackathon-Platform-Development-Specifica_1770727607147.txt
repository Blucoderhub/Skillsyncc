# MASTER PROMPT: Hackathon Platform Development Specification

## Project Overview
Build a comprehensive hackathon hosting platform with educational content, multi-tier authentication, and dynamic pricing based on user location.

## 1. AUTHENTICATION SYSTEM (OAuth Implementation)

### 1.1 User Authentication Methods
- **OAuth 2.0 Providers:**
  - Google OAuth
  - GitHub OAuth
  - LinkedIn OAuth
  - Microsoft Azure AD
  - Email/Password (fallback)

### 1.2 User Types & Access Levels
```
├── Individual Users (Participants)
│   ├── Free Tier
│   └── Premium Tier
├── Organizations (Hackathon Hosts)
│   ├── Startup Plan
│   ├── Business Plan
│   └── Enterprise Plan
└── Admin (Platform Management)
    └── Super Admin Access
```

### 1.3 Authentication Flow
1. User selects login method (OAuth provider or email)
2. Redirect to OAuth provider authorization page
3. Callback with authorization code
4. Exchange code for access token
5. Retrieve user profile information
6. Create/update user in database
7. Generate JWT session token
8. Store session with role-based permissions

### 1.4 Security Requirements
- Implement JWT tokens with refresh mechanism
- Session timeout: 24 hours (configurable)
- Password requirements: min 8 chars, uppercase, lowercase, number, special char
- Two-Factor Authentication (2FA) optional for users, mandatory for admins
- Rate limiting on login attempts (5 attempts per 15 minutes)
- HTTPS only for all authentication endpoints

## 5. DYNAMIC PRICING BY COUNTRY

### 5.1 Location Detection
- **Primary Method:** IP Geolocation API (e.g., MaxMind, ipapi.co)
- **Fallback:** User-selected country during registration
- **Currency Detection:** Auto-detect based on location
- **Tax Calculation:** Apply VAT/GST based on local regulations

### 5.2 Pricing Tiers by Region

**Region-Based Multipliers:**
```javascript
const pricingMultipliers = {
  tier1: {
    // North America, Western Europe, Australia
    countries: ['US', 'CA', 'GB', 'DE', 'FR', 'AU', 'NL', 'SE', 'NO', 'CH'],
    multiplier: 1.0,
    currency: 'USD/EUR/GBP/AUD'
  },
  tier2: {
    // Eastern Europe, South America, Middle East
    countries: ['PL', 'BR', 'MX', 'AR', 'AE', 'SA', 'CZ', 'GR'],
    multiplier: 0.6,
    currency: 'Local Currency'
  },
  tier3: {
    // Asia (excluding high-income), Africa
    countries: ['IN', 'PK', 'BD', 'NG', 'KE', 'ID', 'PH', 'VN', 'EG'],
    multiplier: 0.3,
    currency: 'Local Currency'
  },
  tier4: {
    // Low-income countries
    countries: ['NP', 'LK', 'GH', 'UG', 'MM'],
    multiplier: 0.15,
    currency: 'Local Currency'
  }
};
```

**Base Pricing (USD - Tier 1):**
```
Individual Users:
- Free Tier: $0
- Premium: $9.99/month or $99/year

Organizations:
- Startup: $299/year
- Business: $999/year
- Enterprise: Custom pricing

Hackathon Hosting Fee:
- Platform commission: 5-10% of prize pool
```

### 5.3 Implementation Details
- Display prices in local currency with USD equivalent
- Update exchange rates daily via API (e.g., exchangerate-api.com)
- Show regional pricing disclaimer
- Allow manual country override with verification
- Implement PPP (Purchasing Power Parity) adjustments annually

### 5.4 Payment Gateway Integration
- **International:** Stripe (supports 135+ currencies)
- **India-specific:** Razorpay
- **Additional:** PayPal, cryptocurrency (optional)
- Support for local payment methods (UPI, PIX, etc.)

## 6. DATABASE SCHEMA (High-Level)
```sql
-- Users Table
users (
  id, email, username, password_hash, oauth_provider, 
  oauth_id, role, country_code, currency, created_at, 
  email_verified, two_fa_enabled
)

-- Organizations Table
organizations (
  id, name, website, logo_url, industry, plan_type,
  verified, created_at, country_code
)

-- Hackathons Table
hackathons (
  id, title, description, theme, organization_id,
  start_date, end_date, registration_deadline,
  prize_pool, max_participants, status, created_by
)

-- Learning Content Table
learning_content (
  id, title, slug, content, category, difficulty_level,
  author_id, views, likes, published_at
)

-- Pricing Table
pricing_by_country (
  id, country_code, region_tier, multiplier,
  currency_code, updated_at
)

-- Transactions Table
transactions (
  id, user_id, amount, currency, country_code,
  payment_gateway, status, created_at
)
```
